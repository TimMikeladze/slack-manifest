# slack-manifest-tools

Slack Manifest Tools

## Getting Started

> yarn add slack-manifest-tools --dev

## Usage

```
Usage: cli [options]

Options:
  -m, --manifest <manifest>  app manifest file
  -v, --validate             validate app manifest file
  -u, --update               update app manifest file
  -h, --help                 display help for command
```
